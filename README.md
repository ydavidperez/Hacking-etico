## 🎯 Objetivo del Repositorio

Este repositorio documenta los conceptos clave aprendidos en cada módulo de formación en ciberseguridad ofensiva. El enfoque es **educativo y ético**: comprender cómo piensan los atacantes para construir mejores defensas.

> ⚠️ **Aviso Legal**: Todo el contenido aquí documentado es con fines académicos y de investigación en entornos controlados. El uso de estas técnicas sin autorización explícita es ilegal.

---

## 🧠 Reflexión General

A lo largo de estos módulos se construye una visión integral del proceso de un pentest:

```
Reconocimiento → Ingeniería Social → Explotación de Red → Explotación de Apps
     M3                M4                    M5                    M6
```

Cada fase alimenta a la siguiente: la información recopilada en el M3 permite diseñar ataques de ingeniería social (M4), que a su vez abren puertas a la red (M5) y finalmente a las aplicaciones (M6).
